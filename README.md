# Student Loan Forecaster

A Streamlit app to forecast UK Plan 2 student loan balance with various knobs and dials to play with. Enter your salary, loan balance, and graduation year to simulate monthly repayments over the 30-year loan term, with adjustable parameters for salary growth, interest rates, and voluntary overpayments.

To run locally, install [uv](https://docs.astral.sh/uv/) then run `uv run streamlit run src/app.py` from the project root.
